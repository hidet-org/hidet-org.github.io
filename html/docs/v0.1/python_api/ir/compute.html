class hidet.ir.compute.TensorNode(name, ttype, tensor_compute=None)[source]¶

A node in the compute DAG that represents a tensor.

Can be either an input or an intermediate node. The input node is created by tensor_input(). and the intermediate node is created by compute().

class hidet.ir.compute.ReduceType(value)[source]¶

An enumeration.
